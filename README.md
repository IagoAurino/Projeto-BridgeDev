# 🌉 Projeto BridgeDev

Este é o projeto da plataforma BridgeDev, desenvolvido para conectar soluções tecnológicas com uma interface moderna e intuitiva. O foco principal foi criar um site responsivo e bem estruturado utilizando tecnologias fundamentais da web.

## 🌐 Link do Projeto
🚀 [Clique aqui para acessar o site oficial][https://iagoaurino.github.io/Projeto-BridgeDev/](https://iagoaurino.github.io/Projeto-BridgeDev/]

## 🛠 Tecnologias Utilizadas
* **HTML5:** Estrutura semântica do site.
* **CSS3:** Estilização, layout (Flexbox/Grid) e responsividade.
* **GitHub Pages:** Hospedagem e deploy da aplicação.

## 📁 Como visualizar o projeto localmente
Se quiser testar o código na sua máquina, siga os passos abaixo:
1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
